# Real-time Stock Exchange Dashboard 📈

A financial data application that fetches real-time stock information and provides visual analysis for various ticker symbols. Built with Python, Streamlit, and Yfinance.

## Features
- Real-time stock data fetching.
- Interactive line charts for historical price trends.
- Key financial metrics display (Open, Close, High, Low, Volume).
- User-friendly input for custom stock symbol searches.

## Technologies Used
- **Streamlit**: For the web interface.
- **Yfinance**: To retrieve market data from Yahoo Finance.
- **Plotly/Matplotlib**: For rendering financial charts.
- **Pandas**: For data processing.

## How to run locally
1. Clone the repository.
2. Install dependencies:
   `pip install -r requirements.txt`
3. Launch the app:
   `streamlit run app.py`
